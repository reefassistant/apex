# Neptune Apex Go API Client

[![go-doc](https://pkg.go.dev/badge/go.reefassistant.com/apex?status.svg)](https://pkg.go.dev/go.reefassistant.com/apex)
[![Go Report Card](https://goreportcard.com/badge/go.reefassistant.com/apex)](https://goreportcard.com/report/go.reefassistant.com/apex)
[![GitHub license](https://img.shields.io/github/license/reefassistant/apex)](https://github.com/reefassistant/apex/blob/main/LICENSE)

This package provides a Go API client to interface with your Neptune Apex aquarium controller over your local network.

> :warning: **This API client is not designed nor to be used against Apex Fusion !**
